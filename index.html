Install in seconds

Clone, build, and install with a single script.

$ git clone https://github.com/jurre/huebar.git
$ cd huebar
$ ./scripts/install.sh

Requires macOS 15 (Sequoia) and a Philips Hue Bridge on your network.
Or run from source with swift run.
